   1) hdfs dfs -ls
‚Üí Lists all files and directories in the current local directory.

2)  hdfs dfs -mkdir exp1
‚Üí Creates a new folder named exp1 in the current directory.

3) hdfs dfs -cd exp1
‚Üí Changes the working directory to exp1.

4) hdfs dfs -cat e1
‚Üí Displays the content of the local file e1.

5) hdfs dfs -mkdir guru.txt
‚Üí ‚ùå Wrong usage ‚Äî -mkdir is for directories, not files.
(It should be hdfs dfs -touchz guru.txt to create an empty file.)

6)hdfs dfs -touchz /guru.txt

6) hdfs dfs -touchz /demo1
‚Üí Creates an empty file named /demo1 in HDFS.

7) hdfs dfs -mkdir /demo2
‚Üí Creates a directory named /demo2 in HDFS.

8) hdfs dfs -touchz /demo2/file1.txt
‚Üí Creates an empty file file1.txt inside /demo2 in HDFS.

9) hdfs dfs -put ../exp1/hii /demo2/file3.txt
‚Üí Uploads a local file named hii from ../exp1 to HDFS as /demo2/file3.txt.

10) hdfs dfs -cat /demo2/file3.txt
‚Üí Displays the content of /demo2/file3.txt stored in HDFS.

11) hdfs dfs -get /demo2/file3.txt ../exp1/hii.txt
‚Üí Downloads /demo2/file3.txt from HDFS and saves it locally as hii.txt inside ../exp1.

12) hdfs dfs -moveFromLocal ../exp1/hii.txt /demo2/fromlocaltohdfs.txt
‚Üí Moves the local file hii.txt to HDFS directory /demo2 (deletes from local after moving).
